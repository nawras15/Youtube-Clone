# 🎥 YouTube Clone - Tailwind CSS & YouTube API

A fully responsive YouTube clone application built using modern web technologies. This project fetches real-time data from the official YouTube Data API v3.

## 🚀 Live Demo
[View Live Project Here](nawras-wadi-youtube.netlify.app/)

## ✨ Features
- **Real-time Data:** Fetches trending videos, comments, and suggestions from YouTube.
- **Smart Search:** Search for any video using the integrated search bar.
- **Voice Search:** High-accuracy voice search supporting multiple languages (AR/EN).
- **Dark Mode:** Fully themed dark and light modes for a comfortable viewing experience.
- **Responsive Design:** Optimized for Mobile, Tablet, and Desktop screens.
- **Video Player:** Integrated YouTube player with smooth navigation.

## 🛠️ Technologies Used
- **HTML5** & **CSS3**
- **Tailwind CSS** (Styling)
- **JavaScript (ES6+)** (Logic & API Integration)
- **YouTube Data API v3** (Data Source)
